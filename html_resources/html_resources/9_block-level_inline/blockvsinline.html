Block vs. Inline Elements

Block-Level Elements

Block-level elements start on a new line and take up the full width of their container. They create a "block" of content.

This is a block-level span element inside a div element.

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. They are displayed "inline" with the surrounding content.

For example, this is an inline element within a paragraph of text. It does not create a new line.
