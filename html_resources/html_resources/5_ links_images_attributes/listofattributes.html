List of HTML Attributes

Global Attributes

Global attributes can be used with most HTML elements:

  • class - Specifies one or more class names for an element
  • id - Defines a unique ID for an element
  • style - Inline CSS styles for an element
  • title - Provides extra information about an element (often displayed as a tooltip)

Form-Related Attributes

Attributes commonly used with form elements:

  • name - Name of an input element
  • type - Defines the input type (e .g., text, checkbox, radio)
  • value - Initial value of an input element
  • placeholder - A short hint that describes the expected value of an input field

Image Attributes

Attributes for controlling images:

  • src - Source URL for an image
  • alt - Alternate text for an image (for accessibility)
  • width - Width of the image in pixels
  • height - Height of the image in pixels

Anchor (Link) Attributes

Attributes commonly used with anchor (link) elements:

  • href - URL to which the link points
  • target - Specifies where to open the linked document (e.g., "_blank" for a new tab or window)
  • download - Indicates that the link should be downloaded when clicked
